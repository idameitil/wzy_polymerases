These directories contain GIF images of sugar molecules downloaded from CSDB.

The directory name represents the 'size' of the image, representing the 'scale' parameter
in the CSDB show_snfg URL.

	3/	scale=3 (small images)
	19/	scale=19 (large images)

Within each directory the GIF (image) files are prefixed by the CSDB record_id.
