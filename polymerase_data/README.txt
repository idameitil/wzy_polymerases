polymerases.tsv is a manually edited file, where new annotated polymerases can be added.

add_csdb_and_taxonomy.py retrieves csdb data and taxonomy for each polymerase and creates the file polymerases_with_csdb_and_taxonomy.tsv
 
make_fasta_from_tsv.py makes a fasta file (polymerases.fasta) with all entries in polymerases.tsv
